# AppUsuarios

una aplicación web en Angular para obtener el clima de cualquier ciudad,

## api usada

[weather api](https://openweathermap.org/api)

## Instalación

- clonar el repositorio
- ejecutar el comando `npm install`
- ejecutar el comando `ng serve --o`
